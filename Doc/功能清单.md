# NewLife.IoT 功能清单

> 版本：v3.3 | 日期：2026-07-22 | 🔍审计刷新 + 🧪测试补强 | 测试：219 全通过

本文档追踪 [需求文档](需求文档.md) 各功能的实现/测试/注释三维状态。

## 拆分决策表

本项目为纯 .NET 类库（无前端），无需前后端拆分。各子项目为独立 NuGet 包，按项目模块拆分。

| 需求项 | 原因 | 拆分结果 |
|--------|:----:|---------|
| 网络驱动 | 多协议 | NET-1 IoTTcp + NET-2 IoTUdp + NET-3 IoTHttp |
| 传输层实现 | 多协议 | NET-5a TcpTransport + NET-5b UdpTransport + NET-5c HttpTransport + NET-5d SocketTransport + SER-2 SerialTransport |
